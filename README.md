# Oblivion-Vaults-Minecraft_Mod_Manager
Stop letting your 1.12 mods crash your 1.20 game T-T... OBLIVION is a standalone, aesthetic-focused instance manager for Minecraft. It doesn't just launch the game; it surgically removes your mods, settings, and worlds, locks them in a secure "Vault," and swaps them out instantly. No more manual folder dragging.
